Implementação

Desenvolvemos um circuito completo na protoboard com os LEDs conectados às portas PWM do Arduino para controle de intensidade. Cada botão está mapeado para um LED especifico e um toque unico no buzzer.

O código foi escrito em C++ utilizando a IDE Arduino, implementando:

  • Geração aleatória de sequências
  • Validação da entrada do jogador
  • Sistema de pontuação progressiva (A cada termino de sequência)
  • Feedback visual e sonoro

Desafios

Enfrentamos vários obstáculos durante o desenvolvimento:

  • Montagem dos Componentes: Tentamos Otimizar os Fios para Reduzir ao Máximo
  • Conexões: Tivemos que lidar com conexões instáveis de Jumpers e LED's.
  • Interferência: Em alguns momentos, os componentes interferiram nos sinais dos botões e LEDs.

Aprendizados

Este projeto nos proporcionou conhecimentos valiosos:

  • Princípios de eletrônica digital e analógica
  • Programação de microcontroladores
  • Técnicas de depuração de hardware
  • Trabalho em equipe multidisciplinar
  • Gestão de projeto com versionamento

Conclusão

O projeto foi um sucesso completo, resultando em um produto funcional, divertido e educativo. Demonstramos que é possível criar experiências interativas complexas com componentes acessíveis.

Próximos passos: expandir o projeto com modos de jogo, armazenamento de pontuação e integração com aplicativos móveis.
